# Contributing to Skeleton Health

First off, thank you for considering contributing to Skeleton Health. We're building patient data sovereignty, and every contribution matters.

## Quick Start

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes
4. Push to your fork
5. Open a Pull Request

### How to Sign Off

Add a `Signed-off-by` line to every commit:

```bash
git commit -s -m "Add amazing feature"
```

This automatically adds: `Signed-off-by: Your Name <your.email@example.com>`

PRs without proper sign-off will not be merged.

## Code of Conduct

- Be respectful and constructive
- Focus on what's best for the community and patients
- No harassment, discrimination, or personal attacks
- Report issues to privacy@skeleton.ai-mds.com

## Reporting Security Vulnerabilities

**Do NOT open a public issue for security vulnerabilities.**

Email: privacy@skeleton.ai-mds.com with subject line `[SECURITY]`.

We will acknowledge within 48 hours and provide a fix timeline within 7 days.
